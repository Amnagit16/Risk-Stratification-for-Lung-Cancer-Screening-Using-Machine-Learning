# Lung Cancer Risk Stratification Using Machine Learning

## Project Overview
This project focuses on identifying high-risk individuals for lung cancer screening using data-driven risk stratification. The goal is to support early detection and improve screening efficiency.

---

## Business Problem
Lung cancer has a high mortality rate, but screening everyone is not efficient.  
This project aims to identify high-risk individuals to enable targeted screening.

---

## Dataset Features
- Age and Age Groups  
- Smoking Exposure (Pack-years)  
- COPD Status  
- Family History  
- Risk Score  

---

## Analysis Performed
- Data Cleaning and Preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature Engineering (Risk Score)  
- Risk Stratification Analysis  
- Data Visualization (Bar charts, Heatmaps, Distributions)  

---

## Key Insights
- Cancer risk increases with age  
- Smoking exposure is a major risk factor  
- COPD significantly increases cancer risk  
- Family history has moderate impact  
- Higher risk scores strongly correlate with cancer  

---

## Conclusion
Risk is not uniform across individuals.  
Multi-factor risk stratification enables targeted screening and improves efficiency.
